One page site

Ideal ara negócios de pequeno ou médio porte, o "One Page" é um tipo de site que concentra todas as informações que o visitante precisa em uma única página, de forma objetiva e criativa. Nele, é possível incluir todos os detalhes relevantes sobre sua empresa, produtos ou serviços, de maneira concisa e impactante, proporcionando uma experiência fluida e intuitiva para os usuários.

Wireframe

Um wireframe de um site é um esboço visual básico que representa a estrutura e o layout de uma página da web. É uma representação siplificada e sem detalhes gráficos, que destaca principalmente a disposição dos elementos principais da página, como menus, blocos de conteúdo, imagens e espaços em branco. Para obter um PDF sobre Wireframe clique no botão ou escaneie o QRCode.

HTML5

O HTML, que significa HyperText Markup Language, é uma linguagem fundamental para a criação e exibição de conteúdo na web. Por meio do HTML, os desenvolvedores podem estruturar e organizar elementos de uma página de web de forma lógica e hierárquia.

Box Model CSS

O Box Model é composto por quatro principais propriedades: margen (margin), borda (border), preenchimento (padding) e conteúdo (content). Essas propriedades envolvem e definem a aparência e o espaço ocupado por um elemento HTML.
